Informativa sulla Privacy

Ultimo aggiornamento: 9 Marzo 2026

Questa Informativa sulla Privacy descrive come Bybit Scanner & MultiChart ("l'App") gestisce le informazioni dell'utente.

1. Raccolta dei Dati

L'App è uno strumento di analisi tecnica lato client. Non raccogliamo, memorizziamo né condividiamo dati personali degli utenti su server esterni. L'App interagisce direttamente con le API pubbliche di Bybit per fornire dati di mercato in tempo reale.

2. Utilizzo delle Informazioni

Le uniche informazioni memorizzate sono salvate localmente sul tuo dispositivo (tramite LocalStorage) e includono:

  • Le tue coppie di trading preferite (Pinned Pairs).
  • Le tue preferenze di visualizzazione (Timeframe selezionati).

3. Servizi di Terze Parti

L'App utilizza i seguenti servizi di terze parti:

  • API di Bybit: Per il recupero dei dati di mercato.
  • Lightweight Charts (TradingView): Per la visualizzazione dei grafici.
  • GitHub Pages: Per l'hosting del codice statico dell'App.

Ti invitiamo a consultare le rispettive privacy policy di tali fornitori per ulteriori dettagli.

4. Sicurezza

L'App non richiede credenziali di accesso, chiavi API private o dati finanziari sensibili. Non viene effettuata alcuna transazione finanziaria direttamente all'interno dell'App.

5. Consenso

Utilizzando l'App, acconsenti alla presente Informativa sulla Privacy.
